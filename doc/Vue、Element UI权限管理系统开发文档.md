# Vue、Element UI权限管理系统开发实战

#### 项目地址

http://element-ui.ynitmk.cn/ 

#### 1创建项目

需要下载 

nodejs   vue-cli

###### 1.创建项目命令

```
vue create 项目名称
进项项目配置
cd 项目名称
npm run serve 打开项目
```

#### 2使用git
1. 创建远程仓库
2. 将本地仓库和远程仓库建立连接
3. 将本地仓库的内容推送到远程仓库
4. 添加到暂存区
5. 安装cz-customizable
6. 在package.json进行配置
7. 创建`.cz-config.js`,在这个配置自定义message提交信息规范
8. 使用git cz来进行提交
9. 将本地仓库的代码推送远程仓库
